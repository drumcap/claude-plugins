## 실행 지침

위 컨텍스트를 종합하여 우선순위 할일 목록을 한국어로 출력하라.

긴급도 분류 기준:
- 🔴 **긴급**: D-7 이하 또는 블로커 (당장)
- 🟡 **중요**: D-8~14 (이번 주)
- 🟢 **여유**: D-15 이상 (다음 주)
- 🚨 **블로커**: 다른 작업을 막는 미완료 항목

### 출력 형식

```
✅ 오늘의 할일 체크리스트
================================
오늘: {날짜}

🔴 긴급 - 지금 당장!
  [ ] {작업} (마감 {날짜}, D-{일수})
      → {구체적 실행 방법}

🟡 중요 - 이번 주 안에
  [ ] {작업} (마감 {날짜}, D-{일수})

🟢 여유 - 다음 주까지
  [ ] {작업}

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🚨 블로커 (다른 작업에 영향)
  [ ] {블로킹 항목} → 해결하지 않으면 {영향}

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📊 전체 진행률
  에세이: {완성수}/{전체수}개
  서류:   {완료수}/{전체수}개

🎯 오늘 집중 목표 (1-2개)
  "{가장 임팩트 큰 작업}"

💡 시작:
  /college-transfer:essay {가장 급한 학교}
  /college-transfer:research {리서치 필요한 학교}
```

## 블로커 자동 감지
다음 항목은 🚨 블로커로 표시:
- research/{학교}.md 없이 essay 작성 시도
- 추천인 요청 미완료 (추천서 마감 임박 시)
- 성적증명서 발송 미요청 (서류 마감 D-14 이내)
